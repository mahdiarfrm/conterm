Features

Everything you reach for, faster.

Conterm keeps Ghostty's rock-solid terminal core and adds the chrome a modern macOS app should have.

Splittable panes

Split any pane with ⌘D and ⌘⇧D — recursively, as deep as you like, with spring transitions.

Command palette

Hit ⌘K to fuzzy-search commands, jump to recent paths, search shell history, or switch SSH hosts.

Scrollback search

Search the full scrollback with ⌘F — press Enter to scroll the terminal straight to the match.

SSH-host detection

When you ssh somewhere, the pane chrome switches to the host name — and snaps back the moment you exit.

Tab groups

Organize related tabs into color-coded groups, so a busy window never turns into a guessing game.

Per-window state save

Windows, tabs, panes, and split layouts are all restored exactly where you left them across launches.

Ghostty config compatible

Reads your existing Ghostty config and understands every option — drop in, nothing to relearn.
