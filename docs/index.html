Features

Everything you reach for, faster.

Conterm keeps Ghostty's rock-solid terminal core and adds the chrome a modern macOS app should have.

Splittable panes

Recursive splits with ⌘D and ⌘⇧D, draggable dividers, and ⌥1–⌥9 to focus by number.

Vertical or horizontal tabs

Flip the tab bar to a left sidebar for Arc-style layouts. Auto-hides on left-edge hover when you want the screen back.

Command palette

⌘K fuzzy-searches commands, recent paths, shell history, your SSH config, and every open pane across every window.

Scrollback search

Search the full scrollback with ⌘F — press Enter to scroll the terminal straight to the match.

SSH-host detection

When you ssh somewhere, the pane chrome switches to the host name — and snaps back the moment you exit.

Built-in notes

Quick notes saved with your config and fuzzy-searchable from ⌘K. Capture a snippet without leaving the terminal.

Tab groups

Color-coded grouping for organizing related tabs, browser-style, so a busy window never turns into a guessing game.

460+ themes

Every bundled libghostty theme in a searchable picker with mini-previews. Pick fonts and sizes live.

Live system stats

Optional CPU, RAM, and network sparklines pinned to the tab bar — hover for a popover with detailed graphs.

Liquid Glass chrome

Refractive glass on macOS 26 with a Clear↔Frosted slider, light/dark tint, and a battery-saving mode when backgrounded.

AI agent pills

Per-pane glowing pill shows when Claude Code or opencode is ready, thinking, or needs your attention.

Ghostty-config compatible

Reads your existing Ghostty config and understands every option — drop in, nothing to relearn.
