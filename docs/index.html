A terminal, rethought_

Conterm wraps Ghostty's GPU-accelerated core in Liquid Glass chrome — splittable panes, vertical tabs, tab groups, and a ⌘K command palette that lives over your shell history and SSH config.

Download for macOS View source

Free & open source macOS 14+ Apple Silicon

01 / Splits

Panes that go anywhere.

⌘D opens a pane to the right, ⌘⇧D opens one below. Recurse as deep as you want. ⌥1–⌥9 jumps focus by number. Drag any divider to resize.

03 / Tabs

Top, sidebar, or grouped.

Flip the tab bar to a left sidebar for Arc-style layouts. Auto-hides on left-edge hover when you want the screen back. Tab groups are colour-coded, drag-reorderable, and have their own palette view.

04 / Agents

It knows when Claude is thinking.

Claude Code and opencode are detected per-pane and show a live status pill — ready, thinking, needs you. Non-destructive hook install. Glance across panes without switching focus.

06 / Get it

Drag, drop, done.

Download the latest release, drag it to Applications, you're in. Conterm reads ~/.config/conterm/config so existing Ghostty configs port one-line.
