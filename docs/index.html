04 Agents

Agent command center.

Claude Code and opencode are detected per-pane with a live "ready / thinking / needs you" pill. The command center gathers every running agent across every window into one roster — each with its task, git branch, and live token count and cost — so you can jump to a pane, reply, or interrupt without hunting for it. A dedicated Agents layout turns the sidebar into the roster.
