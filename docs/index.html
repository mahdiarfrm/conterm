Features

What's in 1.0.

Ghostty's terminal core, dressed in a Mac-native chrome you actually want to look at.

Liquid Glass chrome

macOS 26 refractive glass across the window, palette, sidebar, and overlays. Adaptive light or dark tint, frost slider, and a battery-saver that drops to flat fill in the background.

Top or sidebar tabs

Flip orientation any time. The sidebar can auto-hide and slide back in on left-edge hover, with the native traffic lights tucked into a floating glass pill.

Recursive splits

⌘D splits right, ⌘⇧D splits down, any depth. ⌥1–⌥9 jumps focus by number; dividers drag.

Command palette

One launcher for commands, sessions across every window, SSH hosts, shell history, notes, tab groups, and quick "open in Finder / Cursor". Reorderable, keyboard-first.

Tab groups

Color-coded, browser-style groups with inline rename, drag reorder, and a dedicated Groups view in the palette showing every tab in every group.

Session restore

Quit confirmation with a "restore tabs and panes" toggle. Tick it, relaunch, your layout and shells come back exactly where you left them.

Agent status pills

Claude Code and opencode are detected per-pane and show a live "ready / thinking / needs you" pill at the top of the pane. Non-destructive hook install.

SSH-aware chrome

The pane title pill switches to the remote host when you ssh, snaps back to local on exit, and SSH compatibility mode keeps modifier keys working in remote vim and tmux.

One-file config

Conterm reads only ~/.config/conterm/config. A one-line include pulls in your Ghostty config when you want both apps in sync. Backups on every write, live reload in Settings.
